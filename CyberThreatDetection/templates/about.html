Project Overview

AI-Based Cyber Threat Detection Framework

This project implements an intelligent network intrusion detection system using machine learning algorithms. It analyzes network traffic patterns to distinguish between normal connections and potential cyber attacks, providing real-time threat assessment with confidence scoring.

Objectives
  • Detect network intrusions using supervised machine learning
  • Compare multiple ML algorithms and auto-select the best performer
  • Provide both single and batch prediction capabilities
  • Visualize model performance with interactive analytics
  • Maintain prediction audit trails for compliance
  • Deliver a production-ready web application with REST APIs
Dataset

The framework uses a synthetic dataset modeled after the NSL-KDD benchmark dataset — the industry-standard dataset for evaluating network intrusion detection systems. It contains 25,000 records with 41+ features capturing connection-level attributes including:

Basic Features
Duration, protocol, service, flag, bytes transferred
Content Features
Failed logins, root shell, compromised indicators
Traffic Features
Connection counts, error rates, service rates
